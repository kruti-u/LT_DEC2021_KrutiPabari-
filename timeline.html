The timeline and path to learn web development

First few days

Look at different websites online and understand their site design. List down the permanents in website designs and new and creative things that you find. This way you can get to know how to design a basic website and what would it require.

HTML, CSS

To build a website, a simple static with some animations, you will need HTML and CSS. Learn HTML and CSS tags, structure of code, different things you can design using it.

Practice

After you get the gist of basics of HTML and CSS first try making some tables and forms separately and try to do new things in those as you go. Then when you are confident enough try working on a website and include all elements.

Javascript

Javascript is very important for a functional website. You can use functions of it in website to make it more dynamic.

Practice

You'd be capable enough after learning this much to build a dynamic website with good animations and with a better website layout. After this you can intern at different firms to get different projects and challenges to learn more about web development.

Tips

Daily practice is really important when you're trying to learn to code. Made a github profile and share your code their. Never delete a file, you may need it later. That's it! HAPPY LEARNING
